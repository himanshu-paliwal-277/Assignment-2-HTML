List in HTML

  1. There are 3 type of list in HTML
    1. Ordered list
    2. Unordered list
    3. Description list
  2. Ordered list have some order like: numbers,alphabet,roman etc.
    Numbers
    1. one
    2. two
    3. three
    Alphabet
    1. one
    2. two
    3. three
    Roman
    1. one
    2. two
    3. three
  3. Unordered list does not have any order like:
    • 1
    • 2
    • 3
