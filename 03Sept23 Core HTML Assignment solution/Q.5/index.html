Full stack web development tech stack:

Web development
Web development means that creating, buiding and maintenance of website.
Full stack web development
Full stack web development means work on both the frontend and backend of the website.
Frontend
Frontend is a part of webpage that user can see and directly interact and also called clint side of website.
Backend
Backend is a part of webpage that user can not see. It is also called server site which is used to process and store data.

Frontend
HTML :-
HTML is used to structure a website.
CSS :-
CSS is used to design a website.
Bootstap/Tailwind :-
Bootstap & Tailwind are CSS framework.
JavaScript :-
JavaScript is used to give the dynamic behavior to website.
React :-
React is a JavaScript library.
Backend
Node.js :-
Node.js is a JavaScript runtime envoriment used to create a backend of a website.
Express.js :-
Express.js is a Node.js framework.
MongoDB :-
MongoDB is a database used to store a data.
